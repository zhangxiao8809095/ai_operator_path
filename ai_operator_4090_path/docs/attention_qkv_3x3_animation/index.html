视角切换

  1. 整体 O 完整 attention 要填满所有 O 元素
  2. 一个 block 认领一个 O[i,d_out]
  3. 一个线程 只负责部分 j 的贡献
  4. 写回 tid=0 把结果写入 O

颜色含义

Q[i]
当前 query token
K[j]
被扫描的 key token
V[j,d]
当前输出维度的信息
O[i,d]
当前 block 写回的格子

一句话

整体 attention 填满 O；一个 block 填一个 O 格子；block 内线程分工扫描所有 j。
