容易混淆

Block 不是一条线程。Block 是一组能够通过 Shared Memory 和同步屏障协作的线程。

完成动画后记住

CUDA kernel 的三条判断线

01

线程定位

global = blockIdx × blockDim + threadIdx

决定当前线程应该处理哪个数据元素。

02

执行协作

Thread → Warp → Block

Warp 负责硬件执行，Block 提供共享与同步边界。

03

数据放置

Register → Shared → Global

越靠近线程越快、越私有；越远容量越大、共享范围越广。
