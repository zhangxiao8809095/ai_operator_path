attention.cu 的三条主线

01

入口函数不算结果

attention_naive → launch kernel

它负责检查 CUDA、contiguous、float32、形状一致，然后配置 launch。

02

Block 只算一个标量

O[b,h,i,d_out]

blockIdx.x/y/z 分别映射到输出维度、query 位置、batch/head。

03

Softmax 靠两次扫描

max → exp/sum/acc → divide

写法直观，但 QK 重算和内存复用不足，所以很适合作为优化起点。
